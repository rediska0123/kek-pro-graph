# Kek Pro Graph

## Installing

From kek-pro-graph directory:
```
    git submodule update --init --recursive
```

and from kek-pro-graph/serd directory:
```
    ./waf configure
    ./waf
    ./waf install # or sudo ./waf install
```

## Running the tests

```
    make rdf_test
    ./rdf_test
```
